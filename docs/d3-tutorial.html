01

选择与操作 DOM —— d3.select

D3 最基础的操作：选中元素，然后修改它的属性或样式。和 jQuery 很像，但专门为 SVG 设计。

d3.select('#box')           // 选中单个元素（CSS 选择器）
   .attr('fill', '#3b82f6')  // 设置 SVG 属性
   .style('opacity', 0.8)   // 设置 CSS 样式
   .text('Hello D3')         // 设置文本内容

d3.selectAll('circle')      // 选中所有匹配元素
   .attr('r', 20)
💡核心思维：D3 = 选中元素 → 链式修改属性。.attr() 用于 SVG 属性（fill/cx/r），.style() 用于 CSS 属性（color/opacity）。
02

数据绑定 —— .data().join()

D3 最核心的思想：把数据数组绑定到 DOM 元素上，数据有多少条，就自动创建多少个元素。

const data = [30, 70, 50, 90, 40]

svg.selectAll('circle')
   .data(data)          // 绑定数据
   .join('circle')      // 不够就创建，多余就删除
   .attr('cx', (d, i) => i * 60 + 40)  // d=数据值, i=索引
   .attr('cy', 40)
   .attr('r',  d => d / 5)            // 半径 = 数据值 / 5
   .attr('fill', '#3b82f6')
💡核心思维：属性值可以是函数 (d, i) =>，d 是该元素绑定的数据值，i 是索引——这是 D3 和普通 DOM 操作最大的区别。
03

比例尺 —— d3.scaleLinear

比例尺是一个映射函数：把数据范围（domain）映射到像素范围（range）。有了它，不用手算坐标。

// 线性比例尺：数据 0~100 → 像素 0~400
const xScale = d3.scaleLinear()
  .domain([0, 100])   // 输入范围（数据）
  .range([0, 400])   // 输出范围（像素）

xScale(50)  // → 200  （50% 位置）
xScale(25)  // → 100

// 分类比例尺：文字 → 像素位置（做柱状图）
const yScale = d3.scaleBand()
  .domain(['A','B','C'])
  .range([0, 300])
  .padding(0.2)
💡常用比例尺：scaleLinear（数值→数值）、scaleBand（分类→位置，做柱状图）、scaleOrdinal（分类→颜色）、scaleLog（对数）。
04

坐标轴 —— d3.axisBottom

D3 能根据比例尺自动生成坐标轴（刻度线 + 刻度值），不需要手动算每个刻度的位置。

const xAxis = d3.axisBottom(xScale)  // 底部 X 轴
  .ticks(5)                          // 约 5 个刻度
  .tickFormat(d => d + '%')          // 刻度格式化

// 在 g 元素上"调用"轴，D3 自动画出来
svg.append('g')
   .attr('transform', 'translate(0, 100)')
   .call(xAxis)  // ← call() 是关键！把 g 元素传给轴函数

// 还有 axisLeft / axisRight / axisTop
💡.call(axis) 是 D3 特有用法，等价于 axis(selection)。坐标轴样式（颜色/字体）通过 CSS 或 .selectAll('.tick text').style(...) 来改。
05

过渡动画 —— .transition()

在属性修改前加 .transition()，D3 会自动在旧值和新值之间做插值动画，一行代码实现流畅过渡。

// 没有动画（瞬间跳变）
bar.attr('width', 300)

// 加上动画（800ms 缓动过渡）
bar.transition()
   .duration(800)               // 动画时长（毫秒）
   .ease(d3.easeCubicInOut)    // 缓动函数（默认 easeCubic）
   .attr('width', 300)           // 目标值
   .attr('fill', '#10b981')      // 颜色也可以插值！
💡D3 会自动插值数字、颜色、路径（path d 属性），这是 Bar Chart Race 等复杂动画的基础。缓动函数可选：easeLinear / easeBounce / easeElastic 等。
06

事件交互 —— .on()

用 .on(事件名, 回调) 给元素绑定交互，回调参数 (event, d) 中 d 是该元素绑定的数据值，可直接用。

bars.on('mouseover', (event, d) => {
  // d = 这根柱子绑定的数据
  tooltip.style('opacity', 1).text('值：' + d)
})
.on('mouseleave', () => {
  tooltip.style('opacity', 0)
})
.on('click', (event, d) => {
  d3.select(event.currentTarget)
    .transition().duration(300)
    .attr('fill', '#f97316')
})

👆 悬停或点击柱子试试

💡事件回调里 event.currentTarget 是触发事件的 DOM 元素，d3.select(event.currentTarget) 可以立刻修改它。这是 D3 交互最常见的写法。

掌握 D3.js 的学习路径与技术栈

D3 不是孤立的工具，它建立在一系列基础技术之上。下面是从零到能独立开发可视化作品的完整路径。

阶段一 基础前置（必须先掌握） 建议时长：2–4 周
技术 需要掌握到什么程度
HTML / SVG 理解 SVG 基本标签：rect / circle / line / path / text / g，知道坐标系原点在左上角，会写基本 SVG 结构即可
CSS 能写布局、会改 SVG 颜色/字体/透明度，不需要精通动画（D3 自己管动画）
JavaScript（ES6+） 必须熟练：数组方法（map / filter / reduce）、箭头函数、解构赋值、Promise/fetch（加载数据用）。DOM 操作了解即可，D3 会替你做
阶段二 D3 核心入门（能画出基础图表） 建议时长：3–5 周
模块 需要掌握到什么程度
选择 & 绑定 能用 .data().join() 驱动元素创建/更新/删除，理解 enter/update/exit 三态
比例尺 会用 scaleLinear / scaleBand / scaleOrdinal / scaleTime，知道 domain/range 是什么
坐标轴 会生成 X/Y 轴，会自定义刻度格式和样式
过渡动画 会用 .transition().duration().ease()，知道 delay/stagger 错峰动画
事件交互 会做 hover tooltip、click 高亮，理解回调里 (event, d) 的含义
✅ 完成阶段二 = 能独立画出：柱状图、折线图、散点图、饼图
阶段三 进阶技术栈（能做复杂/定制可视化） 按需学习
技术/模块 需要掌握到什么程度
d3-zoom & brush 实现缩放平移、框选过滤，大数据集必用
d3-force 力导向图/网络关系图，掌握各种力（charge/link/collide/center）的调参
d3-geo（地图） 了解投影（projection）、GeoJSON 格式，能画世界地图/省级地图
d3-chord / d3-hierarchy 弦图、树图、旭日图等特殊图表的专用布局算法
Canvas / WebGL 10 万+ 数据点时 SVG 卡顿，需切换到 Canvas；WebGL（regl/Three.js）用于百万级
React/Vue + D3 工程项目中 D3 管数据计算、React/Vue 管 DOM，知道两者如何分工协作
