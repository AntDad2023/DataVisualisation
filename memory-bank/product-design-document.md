# 产品设计文档

> 最后更新：2026-02-10

## 产品定位

数据可视化课程网站——面向数据可视化初学者的教学工具，帮助用户认识常见图表、理解数据与图表的关系、动手生成图表。

## 核心功能

### 1. 图表库（教学）
- 展示 21 种常见图表的教学内容
- 每种图表包含：是什么 / 什么时候用 / 什么时候不该用 / 数据格式要求 / 示例数据
- 支持按 7 类用途筛选：对比、趋势、占比、分布、关系、层级、流向
- 支持从详情页"带入生成器"

### 2. 图表生成器（MVP）
- 支持 8 种图表：条形图、堆叠柱状图、折线图、散点图、饼图、直方图、箱线图、热力图
- 数据输入：CSV 文件上传 / 粘贴表格（Tab 或逗号分隔）
- 自动识别数值列和文本列
- 动态字段映射 UI（根据图表类型显示不同的下拉框）
- ECharts 实时渲染

## 设计风格

- 教学简洁风：白底 + 深灰文字 + 蓝色主色 #2563EB
- 无需登录、无后端、纯前端离线可用

## 页面结构

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 | 课程入口，导航到图表库和生成器 |
| `/charts` | 图表库列表 | 21 种图表卡片 + 分类筛选 |
| `/charts/:id` | 图表详情 | 单个图表的教学内容 |
| `/generator` | 生成器 | 上传/粘贴数据 → 选图表 → 生成 |

## 非目标（当前版本不做）

- 用户账号系统
- 图表导出为 PNG/SVG
- 自定义图表样式（颜色、标题等）
- 后端 API / 数据库
- 移动端适配优化
